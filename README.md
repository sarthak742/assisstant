# Trae Jarvis

This repository contains the Trae Jarvis desktop assistant project. It includes Python modules, a frontend built with web technologies, and build artifacts.

Contents:
- Python modules in `modules/`
- Desktop entry points `jarvis.py` and `desktop.py`
- Frontend in `frontend/`

To initialize locally:

1. Install dependencies for the frontend with npm/yarn inside `frontend/`.
2. Use Python 3.8+ for the backend scripts.
